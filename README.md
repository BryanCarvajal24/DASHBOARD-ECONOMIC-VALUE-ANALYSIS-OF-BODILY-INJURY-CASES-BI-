# DASHBOARD – ECONOMIC VALUE ANALYSIS OF BODILY INJURY CASES (BI)

## Description
Dashboard developed in **Power BI** to analyze the **economic profitability of accident cases**, using **Bodily Injury (BI)** as the primary financial value metric for a law firm in the United States.

The analysis focuses on comparing **paid external case sources** and determining **which sources generate higher real revenue**, beyond the number of cases delivered.

## Key Metrics
- Bodily Injury (BI) as an economic value indicator  
- **Good / Bad / Pending** cases  
- Cases by source  
- Cases by U.S. state  

## Outcome
Enables identification of **which sources provide cases with higher BI and greater revenue potential**, demonstrating that **more cases do not necessarily mean more money**.

## Tool
- Power BI
- Power Query
